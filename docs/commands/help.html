button help

Prints help.

Examples

To get general help:

$ button help

To get help on a specific command:

$ button help build

Alternatively, the --help argument can be specified for a particular command:

$ button build --help

Positional Arguments

  • command

    The command to get help on. If not specified, prints general help.
